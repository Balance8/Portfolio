convertToF

Difficulty: Basic

Prompt

Write a function called convertToF that takes in a temperature in Celsius and converts it to a temperature in Fahrenheit. 

The formula to convert from Celsius to Fahrenheit is: 

Fahrenheit = Celsius*1.8 + 32 

Examples

```
convertToF(0) //=> 32
convertToF(-10) //=> 14
convertToF(10) //=> 50
convertToF(20) //=> 68
