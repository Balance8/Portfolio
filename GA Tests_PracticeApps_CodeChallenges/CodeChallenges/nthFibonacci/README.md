nthFibonacci

Difficulty: Medium

Prompt

Write a function called nthFibonacci that accepts and integer, n and returns the nth Fibonacci number.
The Fibonacci sequence is 0,1,1,2,3,5,8,13,...

Examples

```
nthFibonacci(0); //=> 0
nthFibonacci(1); //=> 1
nthFibonacci(2); //=> 1
nthFibonacci(3); //=> 2
nthFibonacci(4); //=> 3
nthFibonacci(26); //=> 121393
```
